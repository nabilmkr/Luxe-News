# Luxe News

Luxe News is a comprehensive news platform. This repository is a monorepo that contains both the frontend and backend applications for the system.

## Project Structure

*   [`/frontend`](./frontend): The React-based frontend application for the user interface.
*   [`/backend`](./backend): The Express-based backend API for data management and services.
